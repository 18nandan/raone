# Gmail Skill

Allows raone to read, search, and send emails through Gmail.

## When to use
- User asks to check their email
- User asks to find a specific message
- User asks to send an email
- Background: notify user of important emails

## Examples
- "Check my inbox for emails from Alice"
- "Send an email to bob@example.com saying I'll be late"
- "Find the email about the project proposal"
